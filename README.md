# Build
- `cargo build`

# Run
- `cargo run --bin server`
- `cargo run --bin client`

# Play
Currently not alot to do, but getting a base down to build on
- `<UP> <LEFT> <RIGHT> <DOWN>` to move
- `<SPACE>` to jump (currently, this will make you fall through the world)
